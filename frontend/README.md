# Sitzy Frontend

Postaven na React 19, TypeScript, Vite a Tailwind CSS v4.

## Vývoj a testování

```bash
npm run dev       # Spustit Vite dev server (:5173)
npm run build     # Produkční build s PWA manifestem
npm run typecheck # TypeScript validace
npm run lint      # ESLint kontrola
npm run test:e2e  # Spustit Playwright E2E testy
npm run a11y      # Spustit Accessibility (a11y) audit
```

Pro pravidla přístupnosti a auditní workflow viz
[A11Y_GUIDELINES.md](A11Y_GUIDELINES.md).
